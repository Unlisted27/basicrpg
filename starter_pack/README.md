# **BASICS**
starter_pack is a bunch of basic races, professions, and items, all centered around a fantasy theme.

##  HOW ITS ORGANISED
I store everythong as objects (ex: class races():) so that they can be referenced by using . index (ex: races.elf() ).
I do this simply for better looking code, and you are free to store your objects wherever you want
