race

Info:

basicrpg.race objects are necessary for creating basicrpg.character objects. The stat modifiers of a basicrpg.race object will be directly added to the stats of a basicrpg.character object.

Type: basicrpg.race

Variable attributes table

Name Type Description
race.name str The name of the race, will apear in basicrpg.character.printstats()
race.strength_modifier int The strength modifier added directly to the strength stat of a basicrpg.character
race.constitution_modifier int The constitution modifier added directly to the constitution stat of a basicrpg.character
race.intelligence_modifier int The intelligence modifier added directly to the intelligence stat of a basicrpg.character
race.agility_modifier int The agility modifier added directly to the agility stat of a basicrpg.character

Structure:

race(name: str, strength_modifier: int, constitution_modifier: int, intelligence_modifier: int, agility_modifier: int)

Example:

human = basicrpg.race("Human",0,0,0,0)

character

Info:

basicrpg.character objects are fundamental in any RPG game. basicrpg.character objects are coded so that they are not required to be a player, hence the name. This means that every character (player to ant) should be a character object, and thus has the same abilities (from a programming perspective) by default.

Type: basicrpg.character

Variable attributes table

Name Type Description
character.race basicrpg.race The race of the character. Will effect character's stats.
character.profession basicrpg.profession The profession of the character. Shown in basicrpg.character.printstats(). Useless as of now and effects nothing
character.name str The name of the character. Can be anything. You can use basicrpg.genname() to create a random name, but this is not necessary
character.initiative (DEFAULT 0) int Currently unused, could one day effect combat
character.strength (DEFAULT 10) int Strength stat of the character, effected by the character's race's strength_modifier
character.constitution (DEFAULT 10) int Constitution stat of the character, effected by the character's race's constitution_modifier
character.intelligence (DEFAULT 10) int Intelligence stat of the character, effected by the character's race's intelligence_modifier
character.agility (DEFAULT 10) int Agility stat of the character, effected by the character's race's agility_modifier
character.armor_class (DEFAULT 4) int Armor class stat of the character, effected by the character's race's agility_modifier
character.max_health (DEFAULT 10) int Max health stat of the character, effected by the character's race's constitution_modifier
